Navigate to https://www.posti.fi/en/service-points-on-map
1) User should be able to find nearest pickup points using a street address or Zip code (sample data:
Tietajantie4 Espoo, Zip-02130).
2) User can alternatively allow the location services and visualize all Posti service points nearby.
3) While selecting only Postal outlets, search should display only selected information on the map.
4) While using the map, user can hover over the highlighted service points to see the address, opening time,
service facilities and days of service availability.
5) User should see a dynamic view of highlighted service points under the heading Nearest service points along with the map.
6) It should be possible to zoom in and out of the map and visualize it in full screen mode without issues.
